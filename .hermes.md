# forch-i-oracle

## Conexión al Ecosistema FORCH.iA
Este proyecto está conectado al ecosistema FORCH.iA.
El ecosistema se detecta automáticamente buscando AGENTS.md en los directorios padre.
Ruta esperada: `D:/OTRO DISCO/FORCH-IA/FORCH-IA-ECOSYSTEM`

## Cómo Trabajar
1. **Carga el orquestador**: `/skill forchi-orchestrator`
2. **Trabaja en lenguaje natural**: describe lo que necesitas
3. **El agente aplica automáticamente**:
   - Protocolo de 8 fases (Detect → Skill → Rules → Bridges → Memory → Execute → Verify → Learn)
   - TDD antes de código de producción
   - Principios de marca FORCH.iA
   - Verificación antes de declarar completado
4. **Las lecciones se guardan** en el ecosistema para que OpenCode/@forchi también las vea

## Aprendizaje Compartido
Cada vez que aprendes algo valioso, el agente lo guarda en:
`$ECOSYSTEM/.claude/memory/learning/`

Tanto Hermes como OpenCode leen de la misma carpeta.
